# QA Automation Portfolio

## 프로젝트 개요
QA 엔지니어로서 다양한 자동화 테스트 및 모니터링 시스템을 구축한 포트폴리오입니다.

## 프로젝트 목록

### 📡 [api-automation](./api-automation)
**API Health Monitoring & Automation**
- API 엔드포인트 헬스 체크 자동화
- pytest 기반 API 테스트
- Slack 실시간 알림 시스템
- GitHub Actions CI/CD 통합
- 다중 환경 지원 (dev/stg/live)

### 📊 [data-validation](./data-validation)
**Stock Data Validation & Automation**
- Google Sheets 연동 주식 데이터 수집
- 한국/미국 주식 API 통합
- 배치 처리 및 체크포인트 시스템
- 자동화된 데이터 검증 및 업데이트

### 🌐 [playwright-web](./playwright-web)
**Playwright E2E Testing Automation**
- Playwright 기반 웹 UI 자동화
- Page Object Model (POM) 패턴 적용
- 다중 환경 테스트 지원
- Docker 컨테이너화
- Slack 테스트 결과 알림

## 기술 스택

### 언어 & 프레임워크
- **Python 3.9+**
- **pytest** - 테스트 프레임워크
- **Playwright** - 브라우저 자동화

### API & 통합
- **REST API Testing**
- **Google Sheets API**
- **Stock Market APIs**
- **Slack Webhook**

### DevOps & 자동화
- **GitHub Actions** - CI/CD
- **Docker** - 컨테이너화
- **Cron Scheduling** - 정기 실행

### 데이터 & 검증
- **JSON Schema Validation**
- **jsonschema** - 스키마 검증
- **Data Processing & ETL**

## 주요 성과
- 🎯 API 모니터링 자동화로 장애 조기 발견
- 🔄 반복 작업 자동화로 업무 효율성 향상
- 📈 체계적인 테스트 구조로 유지보수성 개선
- 🚀 CI/CD 파이프라인 구축으로 배포 안정성 확보
